Timestamp Microservice

You can write in the link a date or a Timestamp to get a JSON with the Timestamp and the date in the UTC format. Example:

Example Input

url/api/2000-06-17
url/api/28 January 2000
url/api/1535428800000

Example Output

{"unix":1626463574000,"utc":"Fri, 16 Jul 2021 19:26:14 GMT"}
